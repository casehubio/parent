| [cross-foundation-bridge-module-placement.md](cross-foundation-bridge-module-placement.md) | Bridge modules for optional cross-foundation integration live in the event-source repo as an optional submodule, listed before the runtime module in root pom | casehub foundation repos — casehub-engine-ledger, casehub-qhorus-connectors pattern |
| [reactive-pg-devservices-test-profile.md](reactive-pg-devservices-test-profile.md) | Reactive PostgreSQL integration tests use a named %reactive-pg config profile with Dev Services; default %test profile stays H2 in-memory | Any casehubio module with a named datasource adding reactive PostgreSQL integration tests |
