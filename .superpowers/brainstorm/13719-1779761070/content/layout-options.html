A — Right tool window

Channel panel lives on the right, alongside the code. Inline gutter annotations mark reviewed lines. You're always seeing code + review simultaneously. The Claudony panel model, applied to the editor.

B — Bottom panel (cycles)

Review lives at the bottom like a terminal. Each review cycle is a tab — you can see history. The active conversation scrolls in real-time. Familiar to developers who live in the Run/Debug panel. Gutter marks still show in the editor.
